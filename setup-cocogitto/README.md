# Setup Cogogitto Action

Install [Cogogitto](https://github.com/cocogitto/cocogitto) in Github Actions.

## Usage

### Latest Version

```yaml
- uses: armakuni/github-actions/setup-cocogitto@main
```

### Specific Version

```yaml
- uses: armakuni/github-actions/setup-cocogitto@main
  with:
    version: 5.5.0
```
